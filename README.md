# pips-configs
A collection of linux configs, install and utility shell scripts, and cosmetic additions that I commonly use across various distros. 

## How to:
1. Run `install.sh` to install things like programming language essentials and cosmetic terminal tools
2. Add `rc.sh` to the end your current `.*rc` file (such as `.bashrc` or `.zshrc`). Example: `$HOME/repos/pip-configs/rc.sh`.
